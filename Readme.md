
# A Ramen
## 概要
A Ramen はラーメンにAR技術をトッピングしたARアプリケーションになります  
現存するラーメン屋のロゴ、のれん、をARカメラで読み取ることにより、そのラーメン屋のスタンダードなラーメンのモデルが表示されます  
  
ARマーカー作成のためのラーメン屋ののれんやロゴの素材は直接写真を自分自身で撮って使いました
## 開発環境
- unity バージョン2019.1.12f1
- 使用言語：C#  
- vuforia  
- Blender
